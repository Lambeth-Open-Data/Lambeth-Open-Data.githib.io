Welcome to Lambeth's Open Data Platform

Introduction

Welcome to Lambeth Council's open data platform. This platform aims to help direct you to open data currated by Lambeth and other partners, linking together some of the main sources of data across our web site and partners such as Impact on Urban Health and the ONS.

Latest News

New data packs coming soon

With the release of the Lambeth 2030 Borough plan the State of the Borough report will be replaced with a data pack that aligns to the key themes and golden thread of Equity and Justice. The new data pack is expected to be released in early 2025 and will provide data at borough and ward level, where available.

The data pack will amalgamate the State of the Borough, Ward profiler and Cost of Living data packs.

The new data packs will look a little different from the existing report to overcome the load time issues we have faced using the current tools.

  • State of the Borough Report 2023
  • Cost of Living Crisis Data Pack
  • Borough, neighbourhoods and wards *
  • Joint Strategic Needs Assessment
  • Lambeth Health Profiles
  • Lambeth Map
  • Open Mapping Data
  • Ward Profiler, Power BI report
  • Urban Health Index created by Impact on Urban Health
  • ONS Census 2021 Maps and Data

* Previous State of the Borough reports and Resident survey data can be found on this page.
